Ubuntu 开机引导项修改

在安装好 Win 和 Ubuntu 双系统后，有时我们会发现，开一次机需要经历 Win 和 Ubuntu 的两次开机引导，浪费了许多开机时间。

本文的目的在于消灭 Ubuntu 的开机引导，只保留 Win 的开机引导。理由是多年间习惯了 Win 的开机引导样式。任性。:-D

平台版本

Windows 7 x64 ultimate

Ubuntu 15.10

具体操作

在 Ubuntu 终端执行命令：

sudo gedit /etc/default/grub

kylin：

将找到的三个GRUB_TIMEOUT=10中的else上下的两个改为GRUB_TIMEOUT=0.1，注意不可改为0。

原版：

仅有一个GRUB_TIMEOUT=10可供修改。

保存退出，执行命令：

sudo update-grub

这样就可以略过 Ubuntu 的开机引导了。
